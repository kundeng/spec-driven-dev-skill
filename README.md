# spec-driven-dev-skill

`spec-driven-dev` provides a requirements → design → tasks → implement loop for
multi-step work.

This version explicitly supports analytic and data-workflow specs by pairing
with `analytic-workbench` when the project needs notebooks, experiment sweeps,
review gates, or reproducibility promotion.

## Install

```bash
npx skills add https://github.com/kundeng/spec-driven-dev-skill -g --all
```
